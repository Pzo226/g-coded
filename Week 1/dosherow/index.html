The Dev

Open Source

Open source is a strategy of software devolopment in which developers allow others to work on their code, while also adding to or editing other programmers work. All of this is done without any monetary exchange.

Tech Stack

Tech stacks are the full compliment of various technologies that faciliate a given website or application to function. Examples include but are not limited to: programming languages, databases, browsers and operating systems.

Procedural Programming

Procedural programming is a strategy for developing code that implements standard data types and keywords as well as fundamental coding tools like loops and operators.

Object Oriented Programming

Object oriented programming is a strategy for developing code that utilizes objects and methods which programmers can create. This tactic is the the standard in developing software.
