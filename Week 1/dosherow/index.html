The Dev

Open Source

Open source is a strategy of software devolopment in which developers allow others to work on their code, while also adding to or editing other programmers work. All of this is done without any monetary exchange.

Tech Stack

Tech stacks are the full compliment of various technologies that faciliate a given website or application to function. Examples include but are not limited to: programming languages, databases, browsers and operating systems.

Procedural Programming

Procedural programming is a strategy for developing code that implements standard data types and keywords as well as fundamental coding tools like loops and operators.

Object Oriented Programming

Object oriented programming is a strategy for developing code that utilizes objects and methods which programmers can create. This tactic is the the standard in developing software.

Q: Is HTML a programming language?

A: HTML is not a programming language even though the L stands for language. It is a markup language which is used to design and format the content on a webpage.

Q: What is Responsive Designing for Web sites?

A: Responsive website designing allows for web pages to be accessible to the multitude of various devices that might access that site.

Q: Besides Relational what's another type of database?

A: Non relational datatypes are optimized for the specific data being organized and maintained. Examples of non relational datatypes are: MongoDB, Cassandra, HBase, Azure Cosmos DB... ect.

Q: Which programmer makes the most money PHP, Agular, React, or Flutter?

A: All of these positions seem to average around 83k-90k annually, with PHP slightly ahead averaging just under 100k annually.

Q: What's the difference between a IDE and Text editor?

A: An IDE contains everything need for coding such as a text editor, compiler, and debugger. Text editors can only be used to create and edit files.

Q: What's a container?

A: Containers are packages of software that can be easily replicated on various operating systems.
