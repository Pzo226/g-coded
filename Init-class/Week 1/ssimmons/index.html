What is Open Source?

Any program whose source code is made available for use or modification as users or other developers see fit.

What is Tech Stack?

The set of technologies used to develop an application, including programming languages, frameworks, databases, front-end and back-end tools, and APIs.

What is Procedural Programming?

A programming paradigm built around the idea that programs are sequences of instructions to be executed.

What is Object Oriented Programing?

programming paradigm based on the concept of "objects", which can contain data and code. The data is in the form of fields, and the code is in the form of procedures.

Q: Is HTML a programming language?

HTML is NOT a programming language. HTML stands for HyperText Markup Language because it is a markup language.

Q: What is Responsive Designing for Web sites?

Responsive design insures that the styling of the web page will display properly on all screen sizes and devices.

Q: What's the difference between a IDE and Text editor?

Text editors can only write code, while IDE's provide many other features such as previewing and testing code.
